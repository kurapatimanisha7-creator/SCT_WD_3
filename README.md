# SCT_WD_3 - Quiz Game Web Application

This project is developed as part of my Web Development Internship at SkillCraft Technology.

## Project Overview
An interactive multiple-choice Quiz Game Application built using HTML, CSS, and JavaScript. The application displays questions, collects user answers, and shows the final score.

## Features
- Multiple Choice Questions
- Interactive User Interface
- Score Tracking
- Correct/Wrong Answer Highlighting
- Restart Quiz Option
- Responsive Design

## Technologies Used
- HTML
- CSS
- JavaScript

## Learning Outcomes
Through this project, I improved my skills in:
- JavaScript DOM Manipulation
- Event Handling
- Dynamic Content Rendering
- Responsive Web Design
